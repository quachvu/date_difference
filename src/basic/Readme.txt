This package contains 3 sets of test data, one java file and one Readme.txt:
	TestData :
		- DateInput.txt contains all of valid data, adequate for user to run without any errors.
		- DateInput1.txt contains date out of range from 1990 and 2010 date which can generate invalid input format errors
		- DateInput2.txt contains invalid date order, which means date2 later than date1 or not then generating invalid input format errors.

User is able to comment out the code at some first line to use the test data which is desired.
